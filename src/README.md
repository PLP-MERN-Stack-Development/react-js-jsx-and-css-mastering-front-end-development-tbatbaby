# PLP Week 3 Assignment - React Task Manager

A modern task management application built with React.js, Tailwind CSS, and Vite.

## Features
- Task management (Add, Complete, Delete, Filter)
- API integration with JSONPlaceholder
- Responsive design
- Dark/Light theme
- Local storage persistence

## Technologies
- React.js
- Tailwind CSS
- React Router
- Vite

## Setup
npm install
npm run dev
